# HERO.IO
HERO.IO is a web platform similar to the Play Store, designed for users to discover and download a variety of mobile applications. The website allows users to sort apps based on their download counts, making it easy to find the most popular apps. Each application has its own dedicated installation page, providing detailed information and installation instructions. Additionally, the ratings of each app are visually presented in a **chart format**, helping users quickly assess app quality and popularity.

<h2>Main Features</h2>
<ul>
  <li>View app details</li>
  <li>After installation, app is added to the installation page</li>
  <li>App ratings are displayed in chart format</li>
  <li>Apps can be sorted based on download counts</li>
  <li>Apps can be uninstalled from the installation page</li>
</ul>
